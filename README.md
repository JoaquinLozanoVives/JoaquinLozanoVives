## 💼 Proyecto freelance destacado

Proyecto freelance remunerado para un cliente real, centrado en la implementación de una plataforma LMS profesional sobre WordPress.

El proyecto incluyó:

- Instalación y configuración de TutorLMS Pro.
- Integración con WooCommerce.
- Vinculación de productos digitales con cursos online.
- Configuración del flujo de compra y acceso del usuario.
- Trabajo en entorno staging.
- Pruebas funcionales del recorrido completo del usuario.
- Integración de vídeos protegidos mediante Vimeo Business.
- Documentación técnica para el cliente.
